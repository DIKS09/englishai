Создайте Personal Access Token:

1. Откройте: https://github.com/settings/tokens

2. Нажмите: "Generate new token" → "Generate new token (classic)"

3. Note: EnglishAI

4. Expiration: 90 days (или больше)

5. Выберите права:
   ☑ repo (все галочки под repo)

6. Нажмите "Generate token" внизу

7. СКОПИРУЙТЕ ТОКЕН! (показывается только один раз)

8. При загрузке на GitHub вместо пароля введите этот токен


═══════════════════════════════════════════════════════════════════
           📝 РУЧНОЙ СПОСОБ (если скрипт не работает)
═══════════════════════════════════════════════════════════════════

Откройте PowerShell в папке проекта и выполните:

# Очистка
Remove-Item -Recurse -Force .git -ErrorAction SilentlyContinue
Remove-Item -Recurse -Force frontend\.git -ErrorAction SilentlyContinue

# Инициализация
git init

# Настройка
git config user.name "DIKS09"
git config user.email "diks09@github.com"

# Добавление файлов
git add .

# Коммит
git commit -m "Add full EnglishAI project"

# Подключение
git remote add origin https://github.com/DIKS09/englishai.git

# Отправка
git branch -M main
git push -u origin main

═══════════════════════════════════════════════════════════════════
           🆘 ПРОБЛЕМЫ?
═══════════════════════════════════════════════════════════════════

❌ Git не найден?
   → Установите: https://git-scm.com/download/win
   → Перезапустите PowerShell

❌ Authentication failed?
   → Используйте Personal Access Token (см. выше)

❌ Remote origin already exists?
   → git remote remove origin
   → git remote add origin https://github.com/DIKS09/englishai.git
   → git push -u origin main

❌ Permission denied?
   → Убедитесь что вы владелец репозитория DIKS09/englishai
   → Или используйте токен с правильными правами
