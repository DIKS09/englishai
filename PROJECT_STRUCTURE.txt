🎯 КЛЮЧЕВЫЕ КОМПОНЕНТЫ:

Backend API Endpoints:
  POST   /api/essay/generate           → Генерация тем эссе
  POST   /api/dialogue/generate        → Создание диалога
  POST   /api/fill-blank/generate      → Создание упражнений
  GET    /api/history                  → Получение истории
  DELETE /api/history/:id              → Удаление из истории
  GET    /api/health                   → Проверка статуса

🚀 КОМАНДЫ:

Установка:
  npm install                      Установить backend
  cd frontend && npm install       Установить frontend

Разработка:
  npm run dev                      Запустить backend
  npm run client                   Запустить frontend
  npm run dev:all                  Запустить всё ⭐

Проверка:
  npm run check                    Проверить окружение

Production:
  npm start                        Запустить в production
  cd frontend && npm run build     Собрать frontend

📝 ФАЙЛЫ ДЛЯ СОЗДАНИЯ:

Обязательно создайте файл .env в корневой папке:

  MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/englishAI
  OPENAI_API_KEY=sk-your-key-here
  PORT=5000
  NODE_ENV=development

Где взять:
  MongoDB URI:       https://www.mongodb.com/cloud/atlas
  OpenAI API Key:    https://platform.openai.com/api-keys
