📝 ШАГ 1: MONGODB (5 минут)
══════════════════════════════════════════════════════════════
1. https://www.mongodb.com/cloud/atlas/register
2. Создайте кластер (FREE M0)
3. Database Access → Add User
4. Network Access → Add IP (0.0.0.0/0)
5. Connect → Copy строку подключения
6. Замените <password> на ваш пароль

Пример:
mongodb+srv://user:pass123@cluster.net/englishAI?retryWrites=true

══════════════════════════════════════════════════════════════

🔑 ШАГ 2: OPENAI ($5 минимум)
══════════════════════════════════════════════════════════════
1. https://platform.openai.com/signup
2. Billing → Add payment → $5
3. API Keys → Create new → Copy ключ

Пример:
sk-proj-abc123xyz789...

══════════════════════════════════════════════════════════════

💾 ШАГ 3: ФАЙЛ .env (1 минута)
══════════════════════════════════════════════════════════════
Создайте файл .env в корневой папке проекта:

MONGODB_URI=ваш_mongodb_uri_из_шага_1
OPENAI_API_KEY=ваш_ключ_из_шага_2
PORT=5000
NODE_ENV=development
