══════════════════════════════════════════════════════════════════
ШАГ 1/3: MONGODB ATLAS (БЕСПЛАТНО)
══════════════════════════════════════════════════════════════════

📍 Ссылка: https://www.mongodb.com/cloud/atlas/register

1. Зарегистрируйтесь (можно через Google)
2. Create → Build a Database → FREE (M0) → Create
3. Username: englishAI | Password: придумайте (запишите!)
4. Database Access → Add User → Read & Write → Add User
5. Network Access → Add IP → 0.0.0.0/0 → Confirm
6. Database → Connect → Connect your application → Copy строку
7. В строке замените <password> на ваш пароль
8. Добавьте /englishAI перед знаком ?

✅ Готово! Строка выглядит так:
mongodb+srv://englishAI:ВАШ_ПАРОЛЬ@cluster0.abc.mongodb.net/englishAI?retryWrites=true&w=majority


══════════════════════════════════════════════════════════════════
ШАГ 2/3: OPENAI API KEY ($5)
══════════════════════════════════════════════════════════════════

📍 Ссылка: https://platform.openai.com/signup

1. Зарегистрируйтесь (можно через Google)
2. Settings → Billing → Add payment method
3. Пополните баланс: $5 (хватит надолго!)
4. API keys → Create new secret key
5. Имя: EnglishAI → Create
6. СРАЗУ скопируйте ключ! (он больше не покажется)

✅ Готово! Ключ выглядит так:
sk-proj-abc123xyz789...


══════════════════════════════════════════════════════════════════
ШАГ 3/3: СОЗДАЙТЕ ФАЙЛ .env
══════════════════════════════════════════════════════════════════

📝 В КОРНЕВОЙ ПАПКЕ ПРОЕКТА создайте файл с именем: .env

Способ 1 (Блокнот):
  - Откройте Блокнот
  - Вставьте текст ниже (замените на свои данные!)
  - Сохранить как → Имя: .env
  - Тип файла: Все файлы (*.*)

Способ 2 (PowerShell):
  - Откройте PowerShell в папке проекта
  - Выполните: notepad .env
  - Вставьте текст ниже
  - Сохраните

───────────────────────────────────────────────────────────────
СОДЕРЖИМОЕ ФАЙЛА .env:
───────────────────────────────────────────────────────────────

MONGODB_URI=ВСТАВЬТЕ_СЮДА_СТРОКУ_ИЗ_ШАГА_1
OPENAI_API_KEY=ВСТАВЬТЕ_СЮДА_КЛЮЧ_ИЗ_ШАГА_2
PORT=5000
NODE_ENV=development

───────────────────────────────────────────────────────────────

ПРИМЕР ГОТОВОГО ФАЙЛА:

MONGODB_URI=mongodb+srv://englishAI:myPass@cluster.net/englishAI?retryWrites=true
OPENAI_API_KEY=sk-proj-abc123xyz789def456ghi
PORT=5000
NODE_ENV=development

══════════════════════════════════════════════════════════════════
🆘 ПРОБЛЕМЫ?
══════════════════════════════════════════════════════════════════

❌ MongoDB ошибка?
   → Проверьте MONGODB_URI в .env
   → Пароль без спецсимволов?
   → Добавили /englishAI перед ?

❌ OpenAI ошибка 401?
   → Проверьте OPENAI_API_KEY в .env
   → Ключ правильный?

❌ OpenAI ошибка 429?
   → Пополните баланс на platform.openai.com

❌ Порт занят?
   → Закройте другие приложения
   → Или измените PORT в .env

❌ Ничего не работает?
   → Читайте: ФИНАЛЬНАЯ_ИНСТРУКЦИЯ.md
