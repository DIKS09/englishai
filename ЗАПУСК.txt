📚 БЫСТРЫЙ СТАРТ

1️⃣  УСТАНОВИТЕ ЗАВИСИМОСТИ:
   npm install
   cd frontend
   npm install
   cd ..

2️⃣  СОЗДАЙТЕ ФАЙЛ .env В КОРНЕВОЙ ПАПКЕ:
   MONGODB_URI=ваша_строка_подключения_mongodb
   OPENAI_API_KEY=ваш_ключ_openai
   PORT=5000
   NODE_ENV=development

3️⃣  ЗАПУСТИТЕ:
   npm run dev:all

4️⃣  ОТКРОЙТЕ БРАУЗЕР:
   http://localhost:3000

🆘 ПРОБЛЕМЫ?

MongoDB ошибка:
   → Проверьте MONGODB_URI в .env
   → Убедитесь что пароль правильный

OpenAI ошибка 401:
   → Проверьте OPENAI_API_KEY
   → Убедитесь что ключ активен

OpenAI ошибка 429:
   → Пополните баланс на platform.openai.com

Порт занят:
   → Измените PORT в .env

Подробно в файле: SETUP.md (раздел "Решение проблем")
